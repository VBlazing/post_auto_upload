## 生成文章上传的请求数据

需求描述:

1. 在现有代码的基础上更新上传文章方法
2. 保留图片上传和更换文章图片链接的功能

具体步骤:

1. 解析 md 文件，获取文章结尾“数据”代码块中的 json 数据作为请求数据的主体
2. 删除文章的一级标题以及二级标题为“简介”的内容，删除文章顶部的多余空格和换行符
3. 删除文章最后的二级标题为“数据”的内容
4. 将剩余的内容作为 content 插入请求数据中
5. 将请求数据以以 json 的格式存储到最外层的 data 文件中，方便 debug

细节:

1. 注意 json 数据中的字段类型，要准确无误的构造对应的请求数据

输出要求:

1. 对 codex agent 友好
2. 非常详细的步骤拆解
3. 清晰的步骤解释和原理说明
